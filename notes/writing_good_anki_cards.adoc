= Writing good anki cards

== Shorter!

=== Example (Go)

Original:

====
*Go.* An identifier is {{c1::exported}} just if {{c2::its name starts with a capital letter}}.
====

Better:

====
*Go.* An identifier is {{c1::exported}} just if {{c2::it's capitalized}}.
====

== Fit words to world

=== Example (Go)

Original:

====
*Effect* +
copy values from `slice1` to `slice2`, returning the number of elements copied

*How* +
`copy(slice2, slice1)`
====

Better:

====
*Effect* +
copy values to `slice1` from `slice2`, returning the number of elements copied
====

=== Example (Bash)

====
*Effect* +
read the first line from `file`, saving it in `var`

*How* +
`read var file`
====

Better:

====
*Effect* +
read into `var` the first line from `file`
====
